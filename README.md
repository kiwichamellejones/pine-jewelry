# Pine Jewelry

Pine Jewelry is a luxury-inspired jewelry showcase website built with HTML, CSS, and JavaScript. The project was redesigned from an older class build into a more polished, portfolio-ready brand experience focused on visual storytelling, handcrafted detail, and premium presentation.

## Overview

This site presents Pine Jewelry as an artisan brand inspired by nature, heirloom design, and the historic Wild West. The layout was rebuilt to feel more modern, elegant, and editorial while staying fully front-end and easy to deploy on GitHub Pages.

## Features

- Responsive multi-section landing page
- Full-screen hero with strong visual branding
- Clean custom navigation with mobile menu
- Story-driven brand sections
- Collections showcase cards
- Custom order call-to-action section
- Promotional banner
- Email signup form with front-end validation
- Scroll reveal animations
- Built with only HTML, CSS, and JavaScript

## Tech Stack

- HTML5
- CSS3
- JavaScript
- Google Fonts
